\begin{enumerate}
	\item Compile the balance code \shellcmd{make -f Balance.mk\_mpi}
	\item Create directory structure with \texttt{create\_proj\_dir.py} (or by hand)
	\item Change shot and time numbers in template Matlab script of prerun
	\item Run prerun, e.g. when running Matlab in terminal \shellcmd{run script\_prerun.m}
	\item Change balance run template Matlab script and run, e.g. \shellcmd{run script\_timeevol.m}
	\item Post process with Matlab/Python
\end{enumerate}
