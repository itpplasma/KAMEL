# KiLCA_interface

Python interface for KiLCA that simplifies configuration and execution. KiLCA uses plain text configuration files; this interface automates reading and writing these files, allowing parameters to be changed by modifying class variables.

# KiLCA_postprocessor
This is a postprocessing class that handles the output of KiLCA. Most importantly, it handles the EB.dat file that contains the electromagnetic fields. For what this file contains, look in ../../Documentation/kilca-2.pdf. Further, this class includes plotting mehtods.
