# python_utility

## fetch_aug_data_from_list.py
The script fetch_aug_data_from_list.py is useful for fetching fitting data from
the IPP network. It reads in the shot/time values from a text file. Change the 
path accordingly.

## create_proj_dir.py
This python script creates the directory structure used in the balance framework. Before using it, change the project name and parent directory in the script.
